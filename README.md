# 🍿 CGV 상영시간표 디스코드 알림 봇 (CGV Ticket Notifier)

원하는 CGV 극장의 특정 영화 예매가 오픈되는지 실시간으로 모니터링하고, 예매가 열리는 즉시 디스코드(Discord)로 알림을 전송하는 파이썬 기반 자동화 봇입니다.

---

## ✨ 주요 기능 (Key Features)

- **다중 극장 동시 모니터링**  
  용산아이파크몰, 영등포 등 여러 타겟 극장을 리스트에 넣어 동시에 확인할 수 있습니다.

- **특정 영화 타겟팅**  
  특정 영화(예: `프로젝트 헤일메리`)의 예매 오픈만 정확하게 감지합니다.  
  영화 제목을 비워두면 극장 자체의 예매창이 열렸는지 확인하는 **전체 감시 모드**로 동작합니다.

- **스마트 하이브리드 파싱 (상영관/시간표 추출)**  
  복잡한 CGV 웹페이지 텍스트 레이아웃을 분석하여, 단순 오픈 여부를 넘어 오픈된 상영관 이름(`IMAX`, `STARIUM` 등)과 상영 시간표를 텍스트로 깔끔하게 추출해 알림에 포함합니다.

- **디스코드 실시간 웹훅 알림**  
  예매 오픈 시 즉시 접속할 수 있는 **예매 전용 직링크(Direct Link)**와 함께 디스코드 메시지를 전송합니다.

---

## 🛠 요구 사항 (Prerequisites)

- Python 3.x
- Chrome 브라우저 (시스템에 설치되어 있어야 함)
- 디스코드 서버 웹훅 URL (Discord Webhook URL)

---

## 📦 설치 및 실행 (Installation & Usage)

### 1. 패키지 설치

스크립트 실행에 필요한 파이썬 라이브러리들을 설치합니다.

```bash
pip install selenium requests python-dotenv
```

### 2. 환경 변수 설정

프로젝트 루트 디렉토리에 .env 파일을 생성하고 디스코드 웹훅 URL을 입력합니다.

```bash
DISCORD_WEBHOOK_URL=https://discord.com/api/webhooks/당신의_웹훅_주소
```

### 3. 스크립트 설정

코드 상단의 설정 영역에서 원하는 영화 제목, 날짜, 타겟 극장을 환경에 맞게 수정합니다.

```
# ================= 설정 영역 =================
TARGET_MOVIE = '프로젝트 헤일메리'  # 알림을 받을 영화 제목
TARGET_DATE = '20260404'            # 예매를 원하는 날짜 (YYYYMMDD)

TARGET_THEATERS = [
    {"name": "용산아이파크몰", "site_no": "0013"},
    {"name": "영등포", "site_no": "0059"}
]
# =============================================
```

### 4. 봇 실행

설정이 완료되면 스크립트를 실행합니다.
봇이 60초 간격으로 극장을 순회하며 예매 오픈 여부를 확인합니다.
```
python cgv_bot.py
```
